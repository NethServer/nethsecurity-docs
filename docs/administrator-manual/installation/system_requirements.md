# System requirements {#system-requirements-section}

NethSecurity is currently available only for [x86-64 architecture](https://en.wikipedia.org/wiki/X86-64).

Minimum hardware requirements:

- 1 vCPU/cores
- 1GB of RAM
- 1GB of disk space
- 2 Ethernet network cards

Recommended hardware requirements:

- 2 vCPU/cores
- 2GB of RAM
- 1GB of disk space plus an additional disk for persistent data like logs
