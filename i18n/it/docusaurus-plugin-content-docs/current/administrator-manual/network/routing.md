# Route statiche {#static_routes-section}

Il routing statico in Linux consente un controllo preciso su come i pacchetti di dati navigano attraverso una rete. A differenza del routing dinamico, che è automatico, il routing statico richiede una configurazione manuale da parte degli amministratori di rete.

Il routing statico è semplice e prevedibile, il che lo rende ideale per reti stabili in cui i percorsi non cambiano frequentemente. Gli amministratori possono definire le route manualmente, fornendo istruzioni specifiche su come i dati devono viaggiare attraverso la rete.

Le route statiche possono essere configurate dalla pagina `Routes`, nella sezione `Network` dal menu di sinistra.

Per aggiungere una nuova route fare clic sul pulsante **Aggiungi route**. Per modificare una route esistente fare clic sul pulsante **Modifica** nel record della tabella. Per eliminare una route, fare clic sul menu con i tre punti **⋮** nel record della tabella e premere il pulsante **Elimina**.
