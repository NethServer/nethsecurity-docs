# Requisiti di sistema {#system-requirements-section}

NethSecurity è attualmente disponibile solo per [architettura x86-64](https://en.wikipedia.org/wiki/X86-64).

Requisiti hardware minimi:

- 1 vCPU/core
- 1GB di RAM
- 1GB di spazio disco
- 2 schede di rete Ethernet

Requisiti hardware consigliati:

- 2 vCPU/core
- 2GB di RAM
- 1GB di spazio disco più un disco aggiuntivo per dati persistenti come i log
